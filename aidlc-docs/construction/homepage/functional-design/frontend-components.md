# 프론트엔드 컴포넌트 (Frontend Components)

## 공통 레이아웃

### 헤더 (header.jsp)
- 로고 (좌측)
- 5개 메인 메뉴 + 드롭다운 서브메뉴
- 모바일: 햄버거 버튼 → 슬라이드 메뉴
- 스크롤 시 헤더 고정 (sticky)

### 푸터 (footer.jsp)
- 회사명, 사업자번호, 주소, 대표전화
- 개인정보처리방침 링크
- 저작권 표시

## 페이지별 주요 UI 컴포넌트

### 홈 (index.jsp)
- Hero 섹션: 풀스크린 배경 이미지 + 브랜드 슬로건 + CTA 버튼
- 4개 메뉴 진입 카드 (소개/사업/시그니처/주거공간)
- 최신 공지사항 3건 미리보기

### 평면소개 (residence/floorplan.jsp)
- 타입별 섹션을 세로로 나열
- 각 섹션: 타입명 + 면적 + 평면도 이미지
- 상단 앵커 링크로 타입별 빠른 이동

### 공지사항 목록 (support/notice-list.jsp)
- 번호 / 제목 / 작성일 테이블
- 하단 페이지네이션 (이전/다음/번호)
- 제목 클릭 → 상세 페이지

### Q&A 목록 (support/qna-list.jsp)
- 번호 / 제목(비밀글 마스킹) / 작성자 / 답변여부 / 작성일
- 하단 페이지네이션
- "질문하기" 버튼 → 작성 폼

### Q&A 작성 폼
- 닉네임 (선택, placeholder: "익명")
- 비밀글 체크박스 → 체크 시 비밀번호 입력 필드 표시
- 질문 내용 (textarea, 최대 1000자, 글자수 카운터)
- 제출 버튼

### 온라인 문의 (support/contact.jsp)
- 이름 (필수)
- 연락처 (필수, 형식 안내)
- 문의 내용 (textarea, 최대 1000자)
- 개인정보 수집 동의 체크박스 (필수)
- reCAPTCHA v3 (비가시적)
- 제출 버튼

### 구글맵 (business/site.jsp, signature/nearby.jsp)
- Google Maps JavaScript API 임베드
- 사업지 위치 마커
- 주변 시설 페이지: 카테고리별 마커 (병원, 마트, 공원 등)

## JavaScript 모듈

| 파일 | 역할 |
|------|------|
| `main.js` | 드롭다운 메뉴, 모바일 햄버거, 스크롤 이벤트 |
| `contact.js` | 문의 폼 클라이언트 검증, reCAPTCHA 토큰 처리 |
| `qna.js` | Q&A 작성 폼 (비밀글 토글, 글자수 카운터) |
| `gallery.js` | 시그니처 섹션 이미지 슬라이더 |
| `maps.js` | 구글맵 초기화 및 마커 설정 |

## 관리자 페이지 UI

### 공통
- 심플한 관리자 레이아웃 (사이드바 메뉴)
- 외부 CSS 프레임워크 없이 최소한의 스타일

### 공지사항 에디터
- 제목 입력 (text)
- 내용 입력 (textarea 또는 간단한 에디터)
- 공개/비공개 토글
- 저장 / 취소 버튼

### Q&A 답변 폼
- 질문 내용 표시 (읽기 전용)
- 답변 입력 (textarea)
- 저장 버튼
